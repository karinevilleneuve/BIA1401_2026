Chapitre 5 Alignement de séquences

5.1 Pourquoi aligner des séquences

5.2 Objectif de l’exercice

Alignement des séquences du gène ARNr 16S de plusieurs bactéries

Afin de procéder à l’alignement des séquences du gène ARNr 16S de plusieurs bactéries, nous utiliserons R pour extraire, à partir de la base de données SILVA ribosomal RNA database project1, les séquences correspondant aux huit bactéries d’intérêt.

  1. Escherichia coli
  2. Enterobacter cloacae
  3. Staphylococcus epidermidis
  4. Citrobacter youngae
  5. Serratia fonticola
  6. Buttiauxella agrestis
  7. Bacillus subtilis
  8. Acinetobacter calcoaceticus

5.3 Script R

5.3.1 Importer le jeu de données

Encore une fois, la première étape consiste à importer le fichier dans R. Cette fois-ci vous ne travaillez pas avec un tableau de données, mais plutôt avec un fichier en format FASTA. Pour importer le fichier il faut donc utiliser la fonction read.fasta() de la librairie seqinr utilisée précédemment.

Description des paramètres :

  • file = : indique le fichier FASTA à importer ;
  • as.string = TRUE : indique que chaque séquence doit être importée sous forme de chaîne de caractères (character string), plutôt que comme un vecteur de lettres individuelles. Cela facilite souvent la manipulation des séquences par la suite ;
  • forceDNAtolower = FALSE : indique que les nucléotides doivent conserver leur casse d’origine (majuscules/minuscules) ;
  • set.attributes = FALSE : empêche l’ajout automatique d’attributs supplémentaires aux séquences importées permettant d’obtenir une structure de données plus simple et plus facile à manipuler.
Code 
silva = seqinr::read.fasta(file = "https://raw.githubusercontent.com/karinevilleneuve/BIA1401_2026/refs/heads/main/silva_nr99_v138_2_BIA1401.fasta", 
                           as.string = TRUE, 
                           forceDNAtolower = FALSE, 
                           set.attributes = FALSE)


df_silva = t(data.frame(silva))

Vous pouvez visualiser l’objet avec la fonction head(), visualiser l’entête des séquences avec la fonction names(), visualiser une séquence spécifique en spécifiant son numéro de ligne ou nom complet entre crochets [].

5.3.2 Extraire les séquences d’intérêt

L’étape suivante consiste à extraire uniquement les séquences correspondant aux bactéries d’intérêt à partir de l’objet silva.

En observant la sortie de la fonction names(), on remarque que le nom de chaque séquence contient l’information taxonomique associée à la bactérie et que chaque rang taxonomique y est séparé par un point-virgule (;). Il est donc important de respecter cette même nomenclature lors de la recherche des séquences. Vous remarquerez également que nous ne disposons pas du nom taxonomique complet de chaque bactérie d’intérêt, mais plutôt uniquement le nom de l’espèce. L’approche classique en R avec == (option 1 de la section Filtrer les données) exige une correspondance exacte du texte. Cela obligerait à récupérer manuellement l’ensemble du nom complet, ce qui est peu pratique. Pour éviter ce travail manuel, nous utilisons la fonction str_detect() de la librairie stringr, qui permet de détecter la présence d’un motif de texte à l’intérieur d’une chaîne de caractères.

Description des paramètres :

  • silva[] : permet de filtrer la liste en conservant uniquement les séquences pour lesquelles la condition est vraie.
  • str_detect() : fonction pour détecter la présence d’un motif de texte à l’intérieur d’une chaîne de caractères ;
  • names(silva) : indique qu’on recherche le motif dans le nom de chaque séquence de la liste silva ;
  • | : agit comme un opérateur logique OU dans la recherche de texte ;

Résultat attendu :

L’objet nos_sequences contiendra uniquement les huit séquences des bactéries d’intérêt.

Une fois de plus la fonction write.fasta() est utilisée pour enregistrer l’objet en format FASTA.

Code 
library(stringr)

nos_sequences = silva[
  str_detect(
    names(silva),
    "Escherichia;coli|Enterobacter;cloacae|Staphylococcus;epidermidis|Citrobacter;youngae|Serratia;fonticola|Buttiauxella;agrestis|Bacillus;subtilis|Acinetobacter;calcoaceticus")]

write.fasta(nos_sequences, names = names(nos_sequences),nbchar = 80, as.string = TRUE, file.out = "sequence_to_align.fasta")

  1. SILVA est une base de données d’ARN ribosomique dont la dernière version (V138.2-2024) contient plus de 9 millions de séquences. La création de la base de données est le résultat d’une collaboratoire entre le Groupe de génomique microbienne de l’Institut Max Planck de microbiologie marine de Brême (Allemagne), le Département de microbiologie de l’Université technique de Munich et Ribocon.↩︎
