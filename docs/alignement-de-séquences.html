Chapitre 5 Alignement de séquences

5.1 Pourquoi aligner des séquences

Aligner des séquences permet de comparer directement les nucléotides homologues entre plusieurs organismes. Concrètement, l’alignement place les séquences les unes sous les autres de façon à ce que les positions équivalentes sur le plan évolutif se retrouvent dans la même colonne. Dans le contexte du gène ARNr 16S, l’alignement est particulièrement important pour plusieurs raisons :

  1. Comparer les similarités et les différences.

    L’alignement permet d’identifier :

    • les régions conservées (peu variables entre les les espèces) ;
    • les régions variables (propres à certaines espèces ou groupes, elles permettent de les distinguer).

    Ces différences sont utiles pour distinguer les espèces bactériennes et estimer leur degré de parenté

  2. Identifier les bactéries

    Le gène ARNr 16S est couramment utilisé comme marqueur taxonomique en microbiologie. En alignant les séquences de bactéries inconnues avec celles de bactéries de référence (par exemple de la base SILVA), on peut déterminer à quelles espèces ou genres elles ressemblent le plus.

  3. Étudier les relations évolutives

    L’alignement constitue la base nécessaire à la construction d’arbres phylogénétiques. Ces arbres permettent ensuite de visualiser les liens évolutifs entre les bactéries et de voir quelles espèces sont les plus apparentées.

  4. Calculer un pourcentage d’identité

    Une fois les séquences alignées, on peut calculer le pourcentage de similitude entre elles. Le pourcentage est ensuite utilisé pour confirmer l’identification d’une bactérie : par convention, deux souches sont considérées comme appartenant à la même espèce si leur similarité au niveau du gène ARNr 16S est supérieure à 97 %.

5.2 Objectif de l’exercice

Alignement des séquences du gène ARNr 16S de plusieurs bactéries

Dans cet exercice, vous allez utiliser R pour extraire depuis la base de données SILVA ribosomal RNA database project1 les séquences du gène ARNr 16S de huit bactéries d’intérêt, puis les aligner à l’aide de l’outil en ligne MAFFT.

Les huit bactéries étudiées sont :

  1. Escherichia coli
  2. Enterobacter cloacae
  3. Staphylococcus epidermidis
  4. Citrobacter youngae
  5. Serratia fonticola
  6. Buttiauxella agrestis
  7. Bacillus subtilis
  8. Acinetobacter calcoaceticus

5.3 Script R

5.3.1 Importer le jeu de données

La première étape consiste à importer le fichier de séquences dans R. Contrairement aux exercices précédents, vous ne travaillez pas avec un tableau de données, mais avec un fichier en format FASTA. Pour l’importer, utilisez la fonction read.fasta() de la librairie seqinr.

Description des paramètres :

  • file = : indique le fichier FASTA à importer ;
  • as.string = TRUE : importe chaque séquence comme une chaîne de caractères (character string) plutôt qu’un vecteur de lettres individuelles, ce qui facilite la manipulation ;
  • forceDNAtolower = FALSE : conserve la case d’origine des nucléotides (majuscules/minuscules) ;
  • set.attributes = FALSE : empêche l’ajout automatique d’attributs supplémentaires, permettant d’obtenir une structure de données plus simple et plus facile à manipuler.
Code 
silva = seqinr::read.fasta(file = "https://raw.githubusercontent.com/karinevilleneuve/BIA1401_2026/refs/heads/main/data/silva_nr99_v138_2_BIA1401.fasta", 
                           as.string = TRUE, 
                           forceDNAtolower = FALSE, 
                           set.attributes = FALSE)

Explorer l’object importés :

Code 
head(silva) # Affiche les premières séquences
names(silva) # Affiche les en-têtes de toutes les séquences
silva[1401] # Accède à la séquence numéro 1401
silva["Bacteria;Actinomycetota;Actinobacteria;Kitasatosporales;Streptomycetaceae;Streptomyces;mangrovicola"] # Accède à une séquence par son nom complet

5.3.2 Extraire les séquences d’intérêt

L’étape suivante consiste à filtrer l’objet silva pour ne conserver que les huit séquences d’intérêt.

En examinant la sortie de names(silva), vous remarquerez que chaque séquence est identifiée par son nom taxonomique complet, où chaque rang est séparé par un point-virgule (;). L’approche classique qui utilise l’opérateur == (option 1 de la section Filtrer les données) exige une correspondance exacte du texte. L’utilisation de cet opérateur nécessiterait donc de récupérer manuellement l’ensemble du nom complet, ce qui est peu pratique. Pour éviter ce travail manuel, nous utilisons la fonction str_detect() de la librairie stringr, qui permet de détecter la présence d’un motif à l’intérieur d’une chaîne de caractères.

Description des paramètres :

  • silva[] : permet de filtrer la liste en conservant uniquement les séquences pour lesquelles la condition est vraie.
  • str_detect() : fonction pour détecter la présence d’un motif de texte à l’intérieur d’une chaîne de caractères ;
  • names(silva) : indique qu’on recherche le motif dans le nom de chaque séquence de la liste silva ;
  • | : agit comme un opérateur logique ou (or) dans la recherche de texte ;

Résultat attendu :

L’objet nos_sequences contiendra uniquement les huit séquences des bactéries d’intérêt.

Une fois de plus la fonction write.fasta() est utilisée pour enregistrer l’objet en format FASTA.

Code 
library(stringr)

nos_sequences = silva[
  str_detect(
    names(silva),
    "Escherichia;coli|Enterobacter;cloacae|Staphylococcus;epidermidis|Citrobacter;youngae|Serratia;fonticola|Buttiauxella;agrestis|Bacillus;subtilis|Acinetobacter;calcoaceticus")]

write.fasta(nos_sequences, names = names(nos_sequences),nbchar = 80, as.string = TRUE, file.out = "sequence_to_align.fasta")

5.4 Aligner et visualiser avec MAFFT

  1. Rendez-vous sur le site MAFFT

  2. Cliquez sur Choose file et sélectionner le fichier sequence_to_align.fasta

  3. Cliquez sur Submit

  4. Cliquez sur View puis View this MSA in new window

  1. SILVA est une base de données d’ARN ribosomique dont la dernière version (V138.2-2024) contient plus de 9 millions de séquences. La création de la base de données est le résultat d’une collaboratoire entre le Groupe de génomique microbienne de l’Institut Max Planck de microbiologie marine de Brême (Allemagne), le Département de microbiologie de l’Université technique de Munich et Ribocon.↩︎
